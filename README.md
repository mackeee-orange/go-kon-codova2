# go-kon-codova2


## install
    `$npm install`
    これでnode_modulesがインストールされます

    `$ cordova plugin add cordova-plugin-ble-central`
    `$ cordova platform add ios`
    `$ cordova platform add browser`

## preset
    `$npm install -g gulp`
    これでgulpコマンド使えるようになります

## build
    `$gulp browserify`
    `$cordova run [platform]`
